## mern-note-app
A simple note taking app using the MERN stack. This project is primarily for myself to learn and experiment with the MERN web stack.

### Setup
- Make sure Node is installed
- Navigate to backend root directory (backend/)
- Run 'npm install'
- Run 'npm run dev'
